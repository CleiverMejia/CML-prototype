## What does it include?

- data types such as `number`, `string` and `boolean`

- creation of variables: `name = "cleiver";`. You can use underscores, letters, and numbers to name variables; a variable cannot start with a number.

- display messages in the console: `print("Hello world!");`

- conditional if: `if (name == "cleiver") {}`
